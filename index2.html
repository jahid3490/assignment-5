Question-3: What is event delegation in the DOM?

Event Delegation is a technique where a parent element handles events for its child elements using event bubbling. This improves performance and is useful for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event starts from the target element and bubbles up to its ancestors (parent elements) in the DOM hierarchy.
